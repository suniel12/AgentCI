# Quickstart

Get up and running with Agent CI in less than 60 seconds.

## Installation

```bash
pip install agentci
```

## Running the Demo

1. Clone the repo (if you haven't already):
   ```bash
   git clone https://github.com/StartInfininite/AgentCI.git
   cd AgentCI/examples/demo_agent
   ```

2. Run tests:
   ```bash
   agentci run
   ```

3. You should see a green passing test suite!
